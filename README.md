# EffectX (特效X)
服务器场景特效插件，类似于Hypixel游戏结束后的场景特效，提供铁砧雨、全场冻结、飞天猪等场景特效

# 新功能
添加了铁砧雨、全场冻结、昼夜变更、飞天猪、打雷场景特效
后续将持续推出更多的场景特效！

# Q&A
Q：为什么要推出这款插件？  
A：此插件主要是娱乐的，也可以为一些需要的服务器使用。  

Q：类似于Hypixel游戏结束后的场景特效是什么意思？  
A：Hypixel每局小游戏或起床战争游戏结束时会启用场景特效，这款插件相当于模仿了Hypixel的这个功能。  

Q：全场冻结会不会破坏原本的建筑结构？  
A：不会，且在特效结束后会立即还原你原来的建筑，所以不必担心。  

Q：昼夜变更会不会影响游戏原来的时间？  
A：不会，在特效结束后会立即还原你世界原本的时间。  

Q：通过什么方式可以运行场景特效？  
A：目前只能通过指令来运行场景特效，后面会考虑推出Gui。  

Q：为什么不能修改Config？  
A：目前插件暂未推出，后续会推出config.yml、language.yml这类配置文件。

# 支持
Minecraft版本：1.21-1.21.11 (仅测试过1.21.10)    
服务器核心：Paper、Leaves、Leaf  
前置插件：无
